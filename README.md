## How to run

Clone the repo, then run using npm.

```bash
git clone https://github.com/RomuloHe4rt/crud-react.git && cd crud-react
npm install
json-server -p 3001 db.json
npm start
```

To run tests type

```bash
npm test
```

To run eslint checks

```bash
npm run lint
```

To deploy to github pages

```bash
npm run deploy
```
